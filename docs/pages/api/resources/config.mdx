You can retrieve the Loft config through this API.

## Example Config

An example Config:
```yaml
apiVersion: management.loft.sh/v1
kind: Config
metadata:
  creationTimestamp: null
spec: {}
status:
  auth:
    github:
      clientId: my-client-id
      clientSecret: my-client-secret
      redirectURI: https://my-redirect-uri

```
